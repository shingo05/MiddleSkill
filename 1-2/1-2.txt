⚫️課題1−2
① Command + Nを押すと何が起こりますか？
　新規作成、新しいウィンドウを開く


② Command + Shift + 3を押すと何が起こりますか？
　画面全体のスクリーンショット

 
③ Command + Vを押すと何が起こりますか？
　コピー、切り取りしたものを貼り付け


④  Command + Shift + deleteを押すと何が起こりますか？
　ゴミ箱を空にする


⑤Macからログアウトするためのショートカットキーは何ですか？
　Command + shift + Q


⑥画面をロックするためのショートカットキーは何ですか？
　Command + control + Q


⑦保存するためのショートカットキーは何ですか？
　Command + S
